
## 链表操作 本质只有 3 种：
1. 下一步 cur = cur.next
2. 断开 cur.next = null
3. 连接 cur.next = node

## 链表题的 4 大母题（直接背）

如果你能一眼识别你在做哪一类，说明你已经切换成功。

#### 快慢指针（关系型）
1. 环检测
2. 找中点
3. 删除倒数第 k 个
👉 不是算长度，是制造“距离差”

#### 虚拟头结点（结构型）
1. 删除
2. 插入
3. 批量修改
👉 解决“头节点不好处理”的问题

### 原地反转（指针重连）
1. 反转链表
2. k 个一组反转

👉 一次只关心 3 个节点
prev ← cur → next

### 合并 / 拼接（流动型）
1. 合并有序链表
2. 重排链表

👉 像拉拉链一样
